## What is Refact?
Refact is a powerful AI coding assistant that combines a complete toolset for efficient development that comes in two versions:
- Cloud version
- Self-hosted version

For teams, Refact offers an [enterprise edition](https://docs.refact.ai/guides/enterprise/) packed with advanced features.

## Features
Refact includes such features as:
- [Code completion](https://docs.refact.ai/features/code-completion/)
- [Chat](https://docs.refact.ai/features/ai-chat/)
- [Fine-tuning](https://docs.refact.ai/features/fine-tuning/)
- [Self-hosted and Enterprise specific features](https://docs.refact.ai/features/self-hosted/)

## Guides
To get started, follow the guides:
- [Refact for VS Code](https://docs.refact.ai/plugins/vs-code/)
- [Refact for JetBrains](https://docs.refact.ai/plugins/jetbrains/)
- [Self-hosted Refact](https://docs.refact.ai/guides/self-hosted/)
- [Enterprise Refact Edition](https://docs.refact.ai/guides/enterprise/)

## Privacy
Refact allows you to restrict access to particular files or projects, ensuring that your private code or confidential files are protected. We never store your code on the server side.

Read more about privacy settings in the [Privacy docs](https://docs.refact.ai/privacy/).
